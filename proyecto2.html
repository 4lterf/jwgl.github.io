Aplicativo movil en Gestion de Citas

Utilizando Java en Android Studio y PHP para el backend y conexion a MySQL se desarrollo un aplicativo con un login para usuario y doctor que permite al usuario Agendar Citas en una Base de Datos en tiempo real, igualmente se visualiza las citas agendadas ,modifica el perfil del usuario o eliminarlas si es necesario ademas utilizando Firebase se puede enviar una notificacion programada por una cita proxima al dispositivo del usuario.
