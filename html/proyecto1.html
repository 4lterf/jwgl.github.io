Administracion de Usuarios

Este proyecto utilizando HTML,CSS, JavaScript para el fronted , Python con el framework de Flask para el backend y SQL Server como gestor de Base de Datos pude desarrollar un sistema de Administracion de Usuarios en el cual se pueden realizar las siguientes acciones:

  • Crear Usuarios
  • Generar QR en base al id del usuario
  • Buscar Usuarios
  • Eliminar Usuarios
  • Actualizar Usuarios
  • Exportar Datos en PDF
